# IDEATHON Logo Keychain

This project is a 3D-printable keychain design inspired by the **IDEATHON logo**.  
The model was created as a simple, compact, and printable product concept that can be used as an event souvenir, promotional item, or personal 3D printing project.

---

## Project Overview

The aim of this project is to turn the IDEATHON visual identity into a physical and usable keychain model.

The design includes a keyring hole and a smooth structure suitable for 3D printing. It can be printed with common 3D printing materials such as PLA and can be scaled depending on the desired final size.

This project focuses on:

- 3D product design
- Logo-based keychain modeling
- Printable object preparation
- Event souvenir concept development
- Physical product prototyping

---

## 3D Model File

The 3D model file is included in this repository:

```text
ideathon_logo_keychain.3mf
```

The `.3mf` file can be opened and prepared for printing using slicer software such as:

- Bambu Studio
- Ultimaker Cura
- PrusaSlicer
- OrcaSlicer

## Printing Recommendations

Recommended 3D printing settings:

| Setting | Recommendation |
|---|---|
| Material | PLA |
| Layer Height | 0.2 mm |
| Infill | 15% - 20% |
| Wall Count | 3 |
| Supports | Not required depending on orientation |
| Print Orientation | Flat on the build plate |

These settings can be adjusted depending on the printer, filament type, nozzle size, and desired surface quality.

# Infinity Keychain

A simple and smooth infinity-shaped keychain design created for 3D printing.

## Features
- Infinity symbol based design
- Clean and smooth outer edges
- Keyring hole included
- Suitable for everyday use
- Ready for 3D printing

## Files
- `files/infinity_keychain.stl` → ready-to-print STL file
- `files/infinity_keychain.3mf` → editable/print project file (if available)

## Printing Recommendations
- Layer height: 0.2 mm
- Infill: 15%–20%
- Wall count: 3
- Supports: Not required (depending on orientation)
- Material: PLA recommended

## Notes
This design was prepared as a minimal and aesthetic infinity keychain model.
It can be resized depending on your printer and usage preference.
